# AutoRA Workflow

Workflow management tools for AutoRA. 

## User Guide

You will need:

- `python` 3.8 or greater: [https://www.python.org/downloads/](https://www.python.org/downloads/)

Install the workflow package (as part of `autora`):

```shell
pip install --upgrade "autora"
```
